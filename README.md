# CodeAlpha_Real_Estate_Market_Trends – Power BI

## Overview

This project was developed as **Task 3** of the **CodeAlpha Power BI Internship**. The dashboard provides insights into real estate market trends to support investment and development decisions.

##  Objective

To build an interactive Power BI dashboard that analyzes:

* Property prices
* Rental analysis
* Market demand and supply
* Economic indicators
* Geographic market hotspots

## 📊 Dashboard Features

* Property Price Analysis
* Rental Analysis
* Demand vs Supply Analysis
* Market Hotspots (Map Visualization)
* Economic Indicators (Interest Rate, GDP Growth, Inflation)
* Interactive filters and charts

## Tools Used

* Microsoft Power BI
* Microsoft Excel
* DAX (Basic Calculations)

## 📁 Files Included

* `Real_Estate_Market_Trends.pbix`
* `Real_Estate_Market_Trends.xlsx`
* Dashboard Screenshot

## 📈 Key Insights

* Compared property prices across cities.
* Analyzed rental performance for different locations.
* Visualized demand and supply conditions.
* Identified market hotspots using map visualizations.
* Tracked economic indicators influencing the real estate market.
